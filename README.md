# Fortunate Roulette

Fortunate Roulette is a customizable roulette wheel used for whatever you need it for!

Users can create their own list of possible results, spin the roulette, and save/load different custom roulettes.

I am making Fortunate Roulette as a beginner practice project for Boot.dev!
https://www.boot.dev

## Current Features

* Add custom sections to the roulette wheel
* Remove added sections
* Edit existing sections
* View your current roulette
* Spin the roulette and randomly choose a result
* Save and load roulettes as JSON files

## Requirements

* Python 3 installed on your computer

This project currently uses only Python's built-in libraries, so no extra packages are required.

## How to Download from GitHub

### Option 1: Download ZIP

1. Go to the GitHub repository page.
2. Click the green **Code** button.
3. Click **Download ZIP**.
4. Extract the ZIP file.
5. Open the extracted project folder.

### Option 2: Clone with Git

If you have Git installed, open Command Prompt and run:

```cmd
git clone https://github.com/FortuneBela/Fortunate-Roulette
cd Fortunate-Roulette
```

## How to Run

### Option 1: Run with Python

Open Command Prompt in the project folder and run:

```cmd
py main.py
```

If "py" does not work, try:

```cmd
python main.py
``` 

### Option 2: Run with the Batch File

The project includes a `.bat` launcher file. 
You can double-click it from File Explorer to start the program.

The batch file is named `Run Fortunate Roulette.bat`.

## How to Use

When the program starts, you will see a menu:

```text
-~= Fortunate Roulette =~-
1. Add Section
2. Remove Section
3. Edit Section
4. View Current Roulette
5. Spin Roulette
6. Save/Load Roulette
7. Exit
```

Choose an option by typing its number and pressing Enter.

### Saving and Loading

Saved roulettes are stored as ".json" files inside the "Saved Roulettes" folder.

For example, if you save a roulette as:

dinner

it will be saved as:

Saved Roulettes/dinner.json

You can later load it from the Save/Load menu.
